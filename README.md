# Face Recognition System

This project implements a **face detection and recognition system** using Python, OpenCV, and the face_recognition library.

## Features
- Detects faces in images
- Extracts face embeddings
- Recognizes known faces from a database
- Draws bounding boxes and labels on recognized faces
- Supports image upload in Google Colab

## Sample Images
- `sample_images/known_person.jpg`: Known face
- `sample_images/test_image.jpg`: Test image for recognition

## How to Use
1. Open `face_recognition_system.ipynb` in Colab
2. Upload a known person’s image when prompted
3. Upload an image to test recognition
4. The output image will display face rectangles and labels

## Technologies
Python, OpenCV, face_recognition (dlib), NumPy, Matplotlib
